# Демо-сценарий

## Что показать

1. Запуск проекта через Docker Compose.
2. Проверка health endpoint.
3. Swagger по адресу `http://127.0.0.1:8010/docs`.
4. Создание события просмотра.
5. Бронирование места.
6. Повторное бронирование, которое возвращает 409.
7. Просмотр своих бронирований.
8. Отмена брони.
9. Отмена события владельцем.

## Основная мысль для демонстрации

Сервис решает задачу бронирования мест на совместные просмотры фильмов и защищает лимит мест от превышения.
